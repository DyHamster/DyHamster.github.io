# 欢迎使用 Markdown 内容

你现在可以直接把 `.md` 文件放到 `src/content/posts/` 目录中，页面会自动读取并展示：

- 支持 frontmatter（标题、日期、标签、摘要、封面）
- 自动计算阅读时长
- 文章列表与详情页均已接入 Markdown 数据源
